ChCore开发环境搭建

2020-09-15 | ipads

ChCore编译+运行环境(ubuntu 18.04)搭建：

编译：

​ Chcore可以直接使用docker进行编译，安装docker即可。

运行：

​ 需要安装qemu，并支持raspi3，具体步骤如下：

​ 1、首先从qemu官网下载最新版本qemu，解压缩至某一文件夹。

​ 2、先安装几个软件包：

​ sudo apt-get install zlib1g-dev

​ sudo apt-get install libglib2.0-0

​ sudo apt-get install libglib2.0-dev

​ 3、配置qemu，使其支持raspi3

​ sudo ./configure –target-list=arm-softmmu,aarch64-softmmu –enable-debug

​ 4、make

​ 5、make install
